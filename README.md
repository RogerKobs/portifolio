  Nesse repositório está um portifólio sobre meu estudo em React, nele está criado uma tela
como principal, que obtem um gif com o que terá na página, um título e um botão de acesso.
  O portifólio recebera atualizações de acordo com minha aprendizagem e tempo livre para
codar. 
  Cada projeto recebe seu título e será reverenciado nesse arquivo pelo mesmo nome em que
foi criado. Com cada update sendo descrito como um edit (e seu número) contendo as informações
que foram inseridas ou alteradas e uma breve descrição do que foi aprendido.

#Menu principal:

  Este arquivo terá o trabalho em mostrar as atividades desenvolvidas e disponibilizar acesso
a elas.

- Edit 01: Criado um esboço da tela e adicionado o acesso a lista de itens;

#Lista de itens:

  Este arquivo tem a ideia de ser uma atividade onde você criara uma lista, podendo mudar sua cor,
criar grupos, editar e excluir grupos ou informações.

- Edit 01: Criado um esboço de tela, com os botões de mudar a cor dos grupos e um botão para criar
um novo grupo;
